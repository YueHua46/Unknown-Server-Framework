## 專案簡介

Unknown Server Framework（USF）適用於 Minecraft BE 官方伺服器（BDS）、個人存檔、LLSE 和 Realms，定位為生存輔助與伺服器管理插件。

## 功能概覽

USF 已實現傳送系統、群組系統、領地系統、公告功能、管理功能等核心模組。

### 傳送系統

- 返回死亡點
- 玩家 TP
- 固定傳送點（管理員設定）
- 世界共享點（玩家設定）
- 個人傳送點
- 群組共享點
- 隨機傳送
- Home 功能

### 群組系統

- 群組歷史消息
- 群組內聊天
- 領地共享

### 領地系統

- 設定開放成員 / 隊伍
- 設定公有領地

### 公告功能

- 多公告
- 置頂公告
- 新成員公告

### 管理功能

- 背包檢查
- 視角跟蹤
- 管理領地
- 封禁實體
- 編輯管理員
- 編輯封禁名單
- 禁言 / 屏蔽
- 設定頭銜

## 插件設定

- 設定進服提示語
- 設定遊戲輔助功能
- 日誌系統
- 計分板默認值
- 積分系統
- 血量顯示
- 玩家名稱格式修改
- 聊天顯示格式修改
- 傳送點 / 領地 / 群組設定
- 插件內置命令設定
- 插件主選單打開方式設定
